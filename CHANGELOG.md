# Changelog

All notable changes to Claim the Moment are documented here.

## 0.5.9 — 2026-07-21

- Added built-in Help, reversible login onboarding, an in-app credits window, and a prominent user guide.
- Replaced the module control with the Person Rays icon and replaced the GM portrait with openly licensed Daemon Skull artwork.
- Added direct GM handoffs, personal persistent volume, all four configurable shared cues, and live least-to-most roster sorting.
- Made contention and count updates authoritative, acknowledged, retryable, and synchronized across clients.
- Preserved the previous player winner across cancelled rounds so the next automatic fallback remains fair.
- Added state validation and migration, deleted-user pruning, elected multi-GM authority, explicit lifecycle cleanup, and commit-before-audio behavior.
- Improved narrow-window layout, roster accessibility, error recovery, packaging checks, attribution consistency, and automated coverage.

## 0.5.8

- Last published prerelease baseline before the 0.5.9 onboarding and reliability work.
