## Short description

A collaborative Daggerheart spotlight manager: the GM throws focus to the table, players race to claim it, and a least-served fallback keeps every scene moving fairly.

## Full description

Claim the Moment gives a Daggerheart table one shared place to pass narrative focus. The GM can throw the spotlight into a synchronized countdown, take it back immediately, or hand it directly to an eligible player. The first eligible player to answer wins; if nobody claims before time expires, the module fairly selects from the eligible online players with the fewest previous spotlight claims.

Everyone sees the same live roster, totals, contention state, countdown, and winner. Assigned character names and token art appear automatically when available. New users get a role-specific welcome, built-in help, editable keybindings, clear readiness guidance, and a persistent personal volume control.

### Highlights

- Synchronized GM-to-player countdown and first-click claim flow.
- Least-served automatic selection with random tie-breaking.
- GM take, direct handoff, contention toggles, manual totals, and reset controls.
- Shared persistent state that stays synchronized across connected clients.
- Assigned character names and token art in the roster and winner display.
- Configurable cues and GM icon, plus independent per-user module volume.
- Built-in role-specific help, first-login guidance, readiness explanations, and editable keybindings.
- Verified for Foundry VTT 14.364 and Foundryborne Daggerheart 2.5.4.
